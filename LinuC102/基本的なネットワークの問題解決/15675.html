ipコマンドを使って、ARPキャッシュの内容を確認したい。正しいコマンドはどれか

正解

ip neighbor sh

解説

ifconfig、route、arp、netstatなどの旧来のネットワーク関連コマンドを置き換えるために、iproute2 ネットワークユーティリティが開発されました。iproute2を使うことで、ポリシーベースルーティングやトラフィック制御といった高度なネットワーク 設定が可能になります。iproute2ユーティリティでは、ipコマンドでアドレス設定、ルーティング設定などを行います。

ip コマンドの書式は以下の通りです。

ip オブジェクト コマンド [値]

ipコマンドは、オブジェクト（機能）に対して、コマンド（処理）を指定する形式で実行します。各オブジェクトごとに指定可能なコマンドは異なります。また、オブジェクトやコマンドは短縮して指定することも可能です。

その他の選択肢については以下の通りです。
・ip nei cache

neighborオブジェクトのコマンドに「cache」「arp」は存在しないため、誤りです。

・ip arp show
・ip cache show
ipコマンドで扱うオブジェクトには「arp」「cache」は存在しないため、誤りです。

な お、IPv6ではARPの代わりにND（Neighbor Discovery）を使って宛先MACアドレスを入手します。入手したMACアドレスはNDキャッシュに格納されます。NDキャッシュのみを確認する場 合はIPv6を指定する「-6」または「-f inet6」オプションを指定します。

※ポリシーベースルーティング（PBR）：
通常のルーティングは宛先アドレスを元にパケットの送信先を決定しますが、PBRはそれ以外の情報（送信元アドレス、UDP/TCPのプロトコル、ポート番号など）を組み合わせてより詳細にパケットの送信先を決定するルーティング方式です。
