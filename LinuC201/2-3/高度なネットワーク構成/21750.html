以下は netstat -a コマンドで表示された結果の抜粋である。



ここから読み取ることができる情報として正しいものはどれか、すべて選びなさい。

正解

TCP と UDP のどちらにも待ち受けポートが存在する

このホストは Web サーバーとして動作している

解説

netstat コマンドは自ホスト上のネットワークの状態について、様々な情報を表示することができるコマンドです。
ここでは -a オプションを指定して、接続中・待受中のソケットを一覧表示しています。

各部は一番初めの行にある英語の通り、それぞれプロトコル (Proto)、自ホスト上のアドレス (Local Address)、相手のアドレス (Foreign Address)、接続状態 (State) を表しています。

以上を踏まえ、各選択肢を検討します。

・TCP と UDP のどちらにも待ち受けポートが存在する
正しいです。
1-3 行目が TCP での待ち受け、4 行目が UDP での待ち受けになっています。

・このホストは Web サーバーとして動作している
正しいです。
1 行目に http、3 行目に https とあります。これらは Web サービスで利用するプロトコルで、これらポートで接続を待ち受けているため、このホストは Web サーバーであると考えられます。

・現在 SSH のポートで接続が確立されている状態にある
誤りです。
2 行目より ssh のポートでの待ち受けが行われていることがわかりますが、現在接続がされているのかどうかはこの抜粋からは判断できません。
ssh のポートで、かつ State の部分が ESTABLISHED となっている行があれば、接続が確立されていると判断できます。

・http と https のポートは同じプロセスにより待ち受けが行われている
誤りです。
どちらのポートでも待ち受けが行われていることはわかりますが、それらが同一のプロセスによるものかどうかは、この情報だけでは判断できません。

netstat コマンドの表示結果を正しく読み取れるようにしておきましょう。
