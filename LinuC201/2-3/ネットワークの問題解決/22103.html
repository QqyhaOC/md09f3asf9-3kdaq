DNS に対してクエリを発行した際、問い合わせにかかった時間や応答メッセージのサイズを確認できるコマンドは次のうちどれか。

正解

dig

解説

IP アドレスとホスト名の変換を行うコマンドには nslookup や dig があります。
どちらも IP アドレスかホスト名を引数に指定して実行することで、DNS を利用した正引き（ホスト名から IP アドレスへの変換）や逆引き（IP アドレスからホスト名への変換）を行うことができます。
また、nslookup よりも dig の方がより詳しい情報を得ることができます。




上図より、digコマンドは問い合わせにかかった時間（Query time）や、応答メッセージのサイズ（MSG SIZE rcvd）なども表示することが分かります。

したがって正解は
・dig
です。

・nslookup
dig と同じく IP アドレスとホスト名の相互変換を行うコマンドですが、nslookup は 問い合わせにかかった時間や応答メッセージのサイズは表示しませんので、誤りです。

・hostname
自ホストのホスト名やドメインといった情報を参照するためのコマンドですので、誤りです。

・tcpdump
ネットワークインタフェースを監視し、パケットキャプチャを行うためのコマンドですので、誤りです。

・dnsquery
存在しないコマンドです。
